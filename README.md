# FourSpace-client

Frontend of fourspace

## Features

1. Users

    - Login / Logout
    - Signup
    - Profile
    - See list of contests
    - Participate in contest
    - See results
    - See public questions

2. Companies
    - Login / Logout
    - Signup
    - Create contest
    - See Results
    - Profile
    - Add questions
